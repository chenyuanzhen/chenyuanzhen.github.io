蓝桥杯

[蓝桥杯2018初赛]全球变暖

关键字:

BFS + 模拟

该题的基本思路:

  1. 利用BFS确定岛屿及其数量 O(n)O(n)O(n)
  2. 对岛屿进行淹没操作 O(n)O(n)O(n)
  3. 统计剩余的岛屿数量 O(n)O(n)O(n)
  4. 输出答案

该题如果是BFS, 需要注意一个地方, 不然会产生大量重复搜索.

利用BFS一定要先将岛屿标记了, 再放入队列, 避免其余岛屿进行BFS时, 重复放入. 不然, 会有岛屿会被重复放入多次
